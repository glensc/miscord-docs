# Beta-version

If you want to use beta version:

* NPM

  ```text
  [sudo] npm install -g miscord-beta
  ```

* Git

  \`\`\` git pull git checkout dev

## to go back:

git checkout master

\`\`\`

* .zip

Download a .zip from the dev branch [here](../tree/dev)

Currently there are no binaries from beta versions.
